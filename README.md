# DeepTrip AI (ailvxing)

AI 旅行规划应用，概念验证项目。

**技术栈**：React 19 + Vite + TypeScript，集成 Google Gemini（`@google/genai`），lucide-react 图标。

## 本地运行

\\\ash
npm install
npm run dev        # 开发服务器
npm run build      # 生产构建
npm run preview    # 预览构建
\\\

需要在环境变量中配置 Gemini API Key。

## 分支说明

- `main` — 主线
- `master` — DeepTrip AI v2.2.0（首页与旅行管家改版）的独立快照，与 main 无共同历史
- `claude-try` / `codex-try` — 实验分支
